GESS

Gulp, SASS, and Express on an MVC stack.

  • ✔ Initial set-up has Gulp concatenate and minify CSS and JS files. It includes a watch task that refreshes the browser when changes to sccs and js files are made from the public directory, or changes occur in any html file saved in views.
  • ✔ Run 'gulp useref' to update the html, css, and js files in the dist folder.
  • ✔ The Express server initial set-up renders static files from the dist and public folder.

Future commits will include basic configurations for the controllers, models, and config folders and comments on how to get started with SASS
